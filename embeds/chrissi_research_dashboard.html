Quarto + pixi + renv — reproducible manuscript stack
Multi-language environment for manuscript rendering. Python managed by pixi (conda-lock based), R packages by renv, documents compiled by Quarto. Deployed via GitHub Actions.
Stack: Quarto 1.x · pixi (Python env mgr) · renv (R lockfile) · GitHub Actions · GitHub Pages
WSL2 host: RStudio Server on Ubuntu · WSL2
Render trigger: push to main → GHA workflow → quarto render → pages deploy
WSL2 / RStudio Server — R spatial stack
Ubuntu on WSL2 running RStudio Server. Spatial R packages (sf, terra, etc.) required system library resolution for GDAL/PROJ/GEOS dependencies.
System libs resolved: libgdal-dev · libproj-dev · libgeos-dev
