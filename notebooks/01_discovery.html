1.1 Rationale and roadmap

To make the reasoning followable — and to be explicit about where it is speculation versus supported — here is the actual chain of thought, step by step:

  1. Start from a real DNAzyme. I BLASTed the catalytic core of the 10-23 deoxyribozyme (Borggräfe et al. 2022) against the viral nucleotide collection and recovered 16 potyvirus isolates (6 species) carrying that core.
  2. Ask whether it could be active. If the core were catalytically active, it would cleave whatever RNA is complementary to the two flanking oligomers (the arms that give a 10-23 DNAzyme its target specificity). So I searched eukaryotic genomes for perfect matches to the 9 + 9 nt flanks of all 16 isolates — i.e. for the sequences whose transcripts a natural version of this DNAzyme might target.
  3. The flank-matches hinted at an ecological pattern. A quick look at two flanks (from a Turnip mosaic virus and a Maize dwarf mosaic virus isolate) suggested the recovered species might be ecologically associated, which made me speculate the viral↔︎eukaryote matches could be signatures of horizontal gene transfer (HGT) to vectors and/or hosts.
  4. Caveat — this is not yet established. It is not shown that the flank-matching species actually are hosts or vectors of the corresponding potyvirus. I built the BioNexus app (Chapter 7) to test connectivity; the interactions looked sporadic rather than continuous — which is not enough to infer the absence of HGT either.
  5. A small aside. A central target nucleotide was initially omitted; adding it back (as G) gives exactly one isolate a single perfect match to Aphis gossypii (a principal aphid vector) — a nice, suggestive observation, not strong evidence.
  6. The locus is the CI gene. The DNAzyme-like sequence maps to the coding region of the potyviral cylindrical inclusion (CI) protein (Chapters 2–4).

Where this goes. Assessing potential DNAzyme catalytic activity (Chapter 3) shows most homologs carry substitutions at activity-critical core positions, so DNAzyme activity is unlikely / rare — and would additionally require reverse transcription to coincide. The nucleotide-level analyses (logos, GC/CpG, codon usage) are therefore exploratory context, not evidence. The thread that does lead somewhere is the CI protein itself — specifically host-type sorting motifs in the DNAzyme-locus region (a tyrosine-based motif beside an acidic dileucine signal; Chapter 5) — which connects to AP-4-dependent trafficking and the LE/MVB and motivates the immune-trafficking hypothesis of Chapter 9.

1.2 Intro

The starting point is a synthetic 10-23 deoxyribozyme (Borggräfe et al. 2022): a single-stranded DNA able to cleave matching single-stranded RNA. The central catalytic domain of the DNAzyme comprises 15 nt - which are flanked by at least 9 nt 3’ and 5’. The flanks, in turn represent the substrate-binding arms, i.e. they determine the specificity of the DNAzyme towards the RNA it is able to cleave. Complementary base-pairing with target RNA (Watson–Crick), positions the 15-nt catalytic core that cleaves the RNA at an unpaired purine·pyrimidine junction.

  target RNA   3'– A A C C C C A U U | C C A C G U A C A –5'
                   | | | | | | | | |   | | | | | | | | |
  DNAzyme      5'– T T G G G G T A A   G G T G C A T G T –3'
                     binding arm       binding arm
                 catalytic core: 5'– G G C T A G C T A C A A C G A –3'

The RNA is cleaved at the unpaired junction (|) between the two arms. The 15-nt catalytic core GGCTAGCTACAACGA — the part with no base-pairing role in recognition — is the sequence that turns out to match the potyvirus CI coding region (below). Scheme redrawn after Borggräfe et al. 2022 (Borggräfe et al. 2022); no figure is reproduced.

2 Origins

Since I was curious whether this system might also occur in nature, I blasted this 15nt sequence against the viral nucleotide collection on NCBI and found that it matches perfectly to a region in the genomes of several potyviruses.

2.1 Blasting the catalytic center of a DNAzyme

I had blasted the catalytic domain of the 10-23 DNAzyme characterised by Borggräfe et al. (2022).

I was curious, whether they (DNAzyme(s)) might exist or have existed in nature. That’s also why I initially screened the experimental virus database (as opposed to picking one for “higher” organisms) available in NCBI blast. I set the search parameters, so that it worked with short input sequences, not allowing any mismatches or gaps (blast will adjust some settings accordingly when the query is short).

As my query was a short sequence (15mer) and the database is far from small, I had naively expected that I should get some hits by chance alone already, which I suspected could be more or less randomly distributed over a couple of possibly very different (i.e. not necessarily closely related) species.

The search retrieved 16 isolates assigned to a total of 6 species, all belonging to the same genus: Potyvirus. That’s where I almost cut my fingers with Occam’s Razor: Potyviruses are members of the family of Potyviridae, which in turn classify as Type IV group viruses according to the Baltimore classification (i.e. they are +ssRNA viruses). DNAzyme activity therefore is nothing, that could be expected directly.

extract_ncbi_coords <- function(x) {
  nms <- if (methods::is(x, "XStringSet")) names(x) else x
  pattern <- "([A-Z0-9_]+\\.[0-9]+)\\.([+-]):(-?[0-9]+)-(-?[0-9]+)"
  m <- regexec(pattern, nms)
  matches <- regmatches(nms, m)
  do.call(rbind, lapply(matches, function(x) {
    if (length(x) == 5) {
      data.frame(full_name = x[1], seqname = x[2], strand = x[3],
                 start = as.integer(x[4]), end = as.integer(x[5]),
                 stringsAsFactors = FALSE)
    } else {
      data.frame(full_name = paste0(x), seqname = NA, strand = NA,
                 start = NA, end = NA, stringsAsFactors = FALSE)
    }
  }))
}

3.2.1 Implications part I

Now the crazy part. For these two examples (sequences based on a) a Turnip mosaic virus and b) a maize dwarf something virus, the flanking regions appeared to occur MASSIVELY enriched in very specific clades:

  1. Plants.
    1. The fact that the viruses appear to leave traces in the genomes of putative hosts is pretty “wow” already. These viruses are no retroviruses, which in turn are indeed known to insert their genomes or parts of it in that of their hosts. Secondly, because it is not a continuous part of the viral genome, but specifically one that matches a DNAzyme TARGET only (without the catalytic loop thingy in the middle, that still is present in the virus).
    2. Another thing I think is interesting is that when I searched the Turnip virus sequences, I did not find ANY monocotyledons (Turnip is a dicotyledonous plant). When I tried with the Maize flanks, the hits however did indeed include monocotyledonous plants. (And wine btw, and cassava).
  2. Insects
    1. But not just any insects. Pollinators!!!!
    2. This is not only indicating that DNAzymes might have existed in nature, but also that they might have been transferred between species and even kingdoms. Here’s the origin of life … right in front of us.

3.4.1 Some more?

The 10-23 deoxyribozyme has been structurally characterised in a homodimer conformation (Cramer et al. 2023). Its target / catalytic-core arrangement is:

5’ cat 3’
Target a
DNAzyme gcacaatg ggctagctacaacga atccccag

The related 8-17 DNAzyme can operate in a single active structure regardless of the metal-ion cofactor (Wieruszewska et al. 2024). There the catalytic domain corresponds to 5’ GCCAGCGCCTCGAA 3’, and the target harbours an additional nucleotide between the specificity-determining flanks.

3.5.2 The RNA argument

“but the virus is RNA!!?”

An RNA genome is not an obstacle: genome sequences of non-retroviral RNA viruses are widely endogenized into plant genomes (Chiba et al. 2011), providing a precedent for the DNA-level integration of a (+)-strand RNA virus sequence proposed here. As those authors put it:

“The pattern of occurrence of NRVSs and the phylogenetic analyses of NRVSs and related viruses indicate that multiple independent integrations into many plant lineages may have occurred.”

— Chiba et al. (2011)

## Rationale and roadmap {#sec-roadmap}

To make the reasoning followable — and to be explicit about where it is speculation
versus supported — here is the actual chain of thought, step by step:

1. **Start from a real DNAzyme.** I BLASTed the **catalytic core of the 10-23
   deoxyribozyme** [@borggraefe2022] against the viral nucleotide collection and
   recovered **16 potyvirus isolates** (6 species) carrying that core.
2. **Ask whether it could be *active*.** *If* the core were catalytically active,
   it would cleave whatever RNA is **complementary to the two flanking oligomers**
   (the arms that give a 10-23 DNAzyme its target specificity). So I searched
   **eukaryotic** genomes for **perfect matches to the 9 + 9 nt flanks** of all 16
   isolates — i.e. for the sequences whose transcripts a natural version of this
   DNAzyme might target.
3. **The flank-matches hinted at an ecological pattern.** A quick look at two flanks
   (from a *Turnip mosaic virus* and a *Maize dwarf mosaic virus* isolate) suggested
   the recovered species might be **ecologically associated**, which made me
   speculate the viral↔eukaryote matches could be signatures of **horizontal gene
   transfer (HGT) to vectors and/or hosts**.
4. **Caveat — this is not yet established.** It is *not* shown that the
   flank-matching species actually **are hosts or vectors** of the corresponding
   potyvirus. I built the **BioNexus** app (Chapter 7) to test connectivity; the
   interactions looked **sporadic rather than continuous** — which is *not* enough
   to infer the **absence** of HGT either.
5. **A small aside.** A central target nucleotide was initially omitted; adding it
   back (as **G**) gives exactly one isolate a **single perfect match to *Aphis
   gossypii*** (a principal aphid vector) — a nice, suggestive observation, **not**
   strong evidence.
6. **The locus is the CI gene.** The DNAzyme-like sequence maps to the coding region
   of the potyviral **cylindrical inclusion (CI)** protein (Chapters 2–4).

**Where this goes.** Assessing potential DNAzyme catalytic activity (Chapter 3) 
shows most homologs carry **substitutions at activity-critical core positions**, 
so DNAzyme activity is **unlikely / rare** — and would additionally require reverse
transcription to coincide. The nucleotide-level analyses (logos, GC/CpG, codon
usage) are therefore **exploratory context, not evidence**. The thread that *does*
lead somewhere is the **CI protein itself** — specifically **host-type sorting
motifs** in the DNAzyme-locus region (a tyrosine-based motif beside an acidic
dileucine signal; Chapter 5) — which connects to **AP-4-dependent trafficking and
the LE/MVB** and motivates the immune-trafficking hypothesis of Chapter 9.

## Intro

The starting point is a synthetic **10-23 deoxyribozyme** [@borggraefe2022]: a
single-stranded DNA able to cleave matching single-stranded RNA. The central catalytic domain of the DNAzyme comprises 15 nt - which are flanked by at least 9 nt 3' and 5'. The flanks, in turn represent the substrate-binding arms, i.e. they determine the specificity of the DNAzyme towards the RNA it is able to cleave. Complementary base-pairing with target RNA
(Watson–Crick), positions the 15-nt catalytic core that cleaves the RNA at an
unpaired purine·pyrimidine junction. 

```default
  target RNA   3'– A A C C C C A U U | C C A C G U A C A –5'
                   | | | | | | | | |   | | | | | | | | |
  DNAzyme      5'– T T G G G G T A A   G G T G C A T G T –3'
                     binding arm       binding arm
                 catalytic core: 5'– G G C T A G C T A C A A C G A –3'
```

The RNA is cleaved at the unpaired junction (`|`) between the two arms. The
15-nt catalytic core `GGCTAGCTACAACGA` — the part with no base-pairing role in
recognition — is the sequence that turns out to match the potyvirus CI coding
region (below). *Scheme redrawn after Borggräfe et al. 2022 [@borggraefe2022];
no figure is reproduced.*

extract_ncbi_coords <- function(x) {
  nms <- if (methods::is(x, "XStringSet")) names(x) else x
  pattern <- "([A-Z0-9_]+\\.[0-9]+)\\.([+-]):(-?[0-9]+)-(-?[0-9]+)"
  m <- regexec(pattern, nms)
  matches <- regmatches(nms, m)
  do.call(rbind, lapply(matches, function(x) {
    if (length(x) == 5) {
      data.frame(full_name = x[1], seqname = x[2], strand = x[3],
                 start = as.integer(x[4]), end = as.integer(x[5]),
                 stringsAsFactors = FALSE)
    } else {
      data.frame(full_name = paste0(x), seqname = NA, strand = NA,
                 start = NA, end = NA, stringsAsFactors = FALSE)
    }
  }))
}

extract_ncbi_granges <- function(x) {

  GenomicRanges::makeGRangesFromDataFrame(df, keep.extra.columns = TRUE)

### Some more?

The 10-23 deoxyribozyme has been structurally characterised in a homodimer
conformation [@cramer2023]. Its target / catalytic-core arrangement is:

| | 5' | cat | 3' |
|---|---|---|---|
| Target | | a | |
| DNAzyme | gcacaatg | ggctagctacaacga | atccccag |

The related **8-17 DNAzyme** can operate in a single active structure regardless
of the metal-ion cofactor [@wieruszewska2024]. There the catalytic domain
corresponds to 5' GCCAGCGCCTCGAA 3', and the target harbours an additional
nucleotide between the specificity-determining flanks.

### The RNA argument

> "but the virus is RNA!!?"

An RNA genome is not an obstacle: genome sequences of non-retroviral RNA viruses
are **widely endogenized into plant genomes** [@chiba2011], providing a precedent
for the DNA-level integration of a (+)-strand RNA virus sequence proposed here.
As those authors put it:

> "The pattern of occurrence of NRVSs and the phylogenetic analyses of NRVSs and
> related viruses indicate that multiple independent integrations into many plant
> lineages may have occurred."
