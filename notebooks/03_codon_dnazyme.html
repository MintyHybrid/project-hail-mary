3.3 Predicted DNAzyme Activity

The DNAzyme-like locus matches the catalytic core of the 10-23 RNA-cleaving deoxyribozyme (15 nt, GGCTAGCTACAACGA) (Santoro and Joyce 1997). Whether a given homolog would retain catalytic activity depends on conservation of this core: it is highly intolerant to substitution (Zaborowska et al. 2002), and position G14 is the catalytic general base that activates the 2’-OH nucleophile (Borggräfe et al. 2022; Cramer et al. 2023). Below, each homolog’s best-matching core window is scored against the canonical core with per-position criticality weights (a sequence-based heuristic, not a catalytic-rate measurement).

3.10.2 Dinucleotide Bias (CpG suppression)

RNA viruses of plants and animals are frequently depleted in the CpG (and often UpA) dinucleotide relative to what their base composition predicts — a signature associated with evasion of host nucleic-acid immune sensing (Greenbaum et al. 2008; Takata et al. 2017). The odds ratio \(\rho = f_{XY} / (f_X f_Y)\) measures observed-vs-expected dinucleotide frequency: \(\rho < 1\) indicates suppression, \(\rho > 1\) over-representation.

3.12 Rolling genome-wide composition

A sliding window along the polyprotein alignment maps how compositional bias varies from region to region, rather than as a single genome-wide average. Local dips in GC3 (third-codon-position GC) track translational/codon adaptation, troughs in CpG O/E localise immune-evasion signal to particular genes, and Shannon entropy marks the most conserved (low-entropy) versus most variable segments. Windows are computed with potytools::rolling_alignment_stats() (300 bp window, 30 bp step, codon-aware).

## Phylogenetic Analysis {.tabset}

```{r phylogenetic-setup}
#| cache: true
#| include: false

# Neighbour-joining trees for each region with bootstrap support. Uses the raw
# p-distance (bounded, robust to the sequence saturation that makes corrected
# distances like TN93 diverge to Inf on the deeper CI/polyprotein alignments)
# and NJ (njs tolerates incomplete matrices). Requires phangorn; freeze caches
# the result so it is computed once.
run_phylo <- requireNamespace("phangorn", quietly = TRUE)
n_boot    <- 30

trees            <- NULL
tree_comparisons <- list()

if (run_phylo) {
  pp_dna <- readDNAStringSet(here::here("data", "claude_polyproteinDNA_mafft.fasta"))
  names(pp_dna) <- gsub(" .*", "", names(pp_dna))

  ci_cds <- readDNAStringSet(here::here("data", "claude_ci2cdsaln.fasta"))
  names(ci_cds) <- gsub(" .*", "", names(ci_cds))

  regions <- list(motif = motif_seqs, ci_cds = ci_cds,
                  pp_dna = pp_dna, DNAzyme = DNAzyme_seqs)

  trees <- list()
  for (region_name in names(regions)) {
    cat(sprintf("\n=== Building tree for %s ===\n", region_name))
    tree <- tryCatch(
      build_phylo_tree(regions[[region_name]],
                       method = "raw", tree_method = "nj", bootstrap = n_boot),
      error = function(e) { message("  skipped ", region_name, ": ", conditionMessage(e)); NULL })
    if (!is.null(tree)) trees[[region_name]] <- tree
  }
  if (length(trees) == 0) trees <- NULL
}
```

```{r tree-comparisons}
#| include: false
#| cache: true

if (!is.null(trees) && length(trees) > 0) {
  region_names <- names(trees)
  for (i in 1:(length(region_names) - 1)) {
    for (j in (i + 1):length(region_names)) {
      region1 <- region_names[i]; region2 <- region_names[j]
      comp    <- paste(region1, region2, sep = "_vs_")
      cat(sprintf("\nComparing %s:\n", comp))
      tree_comparisons[[comp]] <- compare_trees(trees[[region1]], trees[[region2]])
    }
  }
}
```

```{r cu-volcano}
#| fig.height: 6
#| fig.width: 8
#| fig.cap: "Codon-usage volcano. Each point is a sense codon; the x-axis is the log2 fold-change of its usage frequency in the DNAzyme-matching motif relative to the genome background, the y-axis the significance of a Fisher test (Benjamini-Hochberg adjusted). Codons past the dashed thresholds (adjusted p < 0.05 and > 1.5-fold) are coloured by direction and labelled."
volcano_codon <- function(cmp, title, padj_thr = 0.05, lfc_thr = log2(1.5)) {
  d <- as.data.frame(cmp)
  # compare_codon_usage_fast returns 1D arrays; coerce to plain columns on the
  # single data frame we plot from (keeps row order; no case_when/ifelse).
  d$log2FC   <- as.numeric(log2(d$fold_change))
  d$nlt_padj <- as.numeric(-log10(d$p_adjusted))
  # RNAseq-style category assignment by row index; NA p-values stay "n.s.".
  pass <- !is.na(d$p_adjusted) & d$p_adjusted < padj_thr
  d$cat <- "n.s."
  d$cat[pass & d$log2FC >  lfc_thr] <- "Enriched in motif"
  d$cat[pass & d$log2FC < -lfc_thr] <- "Depleted in motif"
  d$cat <- factor(d$cat, levels = c("Enriched in motif", "Depleted in motif", "n.s."))
  d$sig <- d$cat != "n.s."
  ggplot(d, aes(log2FC, nlt_padj)) +
    geom_vline(xintercept = c(-1, 1) * lfc_thr, linetype = "dashed", colour = "grey60") +
    geom_hline(yintercept = -log10(padj_thr), linetype = "dashed", colour = "grey60") +
    geom_point(aes(colour = cat), alpha = 0.85, size = 2.6) +
    geom_text(data = d[d$sig, , drop = FALSE],
              aes(label = paste0(codon, " (", amino_acid, ")")),
              size = 3, vjust = -0.8, check_overlap = TRUE) +
    scale_colour_manual(values = c("Enriched in motif" = "#c0392b",
                                   "Depleted in motif" = "#2980b9",
                                   "n.s." = "grey75"), name = NULL) +
    labs(title = title,
         x = expression(log[2] ~ "fold-change (motif / reference)"),
         y = expression(-log[10] ~ "adjusted " * italic(p))) +
    theme_cowplot() + theme(legend.position = "top")
}
volcano_codon(cu_comparison, "Codon usage: DNAzyme motif vs genome background")
```

## Codon Co-occurrence Network

Beyond per-codon frequencies, the *architecture* of codon usage — which codons
tend to sit next to which — carries additional signal (codon-pair bias). The
network below treats each sense codon in the DNAzyme-locus window as a node and
draws a weighted edge between codons that occur adjacently, pooled across all
isolates. Node colour encodes the amino acid, node size the degree, and Louvain
communities highlight groups of codons that recur together.

```{r codon-network}
#| fig.height: 7
#| fig.width: 9
#| fig.cap: "Codon co-occurrence network of the DNAzyme-locus window. Nodes are sense codons (colour = amino acid, size = degree); edges connect adjacent codons, weighted by how often the pair occurs across all isolates."
suppressMessages({
  library(igraph)
  library(ggraph)
})
gcode <- Biostrings::GENETIC_CODE

pair_tab <- table(unlist(lapply(as.character(motif_seqs), function(s) {
  cds <- potytools::extract_codons(s)
  if (length(cds) < 2) {
    return(character(0))
  }
  vapply(seq_len(length(cds) - 1L),
    function(i) paste(sort(c(cds[i], cds[i + 1L])), collapse = "-"),
    character(1))
})))
pair_tab <- pair_tab[pair_tab >= 8]

edf <- as.data.frame(do.call(rbind, strsplit(names(pair_tab), "-")),
  stringsAsFactors = FALSE)
names(edf) <- c("from", "to")
edf$weight <- as.integer(pair_tab)

g <- igraph::graph_from_data_frame(edf, directed = FALSE)
igraph::V(g)$amino_acid <- unname(gcode[igraph::V(g)$name])
igraph::V(g)$degree <- igraph::degree(g)
igraph::V(g)$community <- as.factor(igraph::membership(igraph::cluster_louvain(g)))

ggraph(g, layout = "fr") +
  geom_edge_link(aes(width = weight), colour = "grey75", alpha = 0.5) +
  geom_node_point(aes(size = degree, colour = amino_acid)) +
  geom_node_text(aes(label = name), size = 2.6, repel = FALSE, vjust = -1) +
  scale_edge_width(range = c(0.3, 2.5), guide = "none") +
  scale_size(range = c(2, 9), guide = "none") +
  labs(colour = "Amino acid") +
  theme_graph(base_family = "sans") +
  theme(legend.position = "right")
```

### Dinucleotide Bias (CpG suppression)

RNA viruses of plants and animals are frequently depleted in the **CpG** (and
often **UpA**) dinucleotide relative to what their base composition predicts — a
signature associated with evasion of host nucleic-acid immune sensing
[@greenbaum2008; @takata2017]. The odds ratio $\rho = f_{XY} / (f_X f_Y)$
measures observed-vs-expected dinucleotide frequency: $\rho < 1$ indicates
suppression, $\rho > 1$ over-representation.

```{r dinucleotide-oe}
#| fig.height: 5
#| fig.width: 10
#| fig-cap: "Dinucleotide odds ratios (ρ = observed/expected) across the DNAzyme-matching motif locus. Bars below the dashed line (ρ = 1) are suppressed; CpG is highlighted. Marked CpG suppression is consistent with an immune-evasion signature."
dinuc <- calculate_dinucleotide_freq(motif_seqs)
dinuc$highlight <- ifelse(dinuc$dinucleotide == "CG", "CpG", "other")

ggplot(dinuc, aes(x = reorder(dinucleotide, rho), y = rho, fill = highlight)) +
  geom_col(alpha = 0.85) +
  geom_hline(yintercept = 1, linetype = 2) +
  scale_fill_manual(values = c(CpG = "#D62828", other = "#2E86AB"),
                    name = NULL) +
```

## Rolling genome-wide composition {.tabset}

A sliding window along the polyprotein alignment maps how compositional bias
varies from region to region, rather than as a single genome-wide average. Local
dips in **GC3** (third-codon-position GC) track translational/codon adaptation,
troughs in **CpG O/E** localise immune-evasion signal to particular genes, and
**Shannon entropy** marks the most conserved (low-entropy) versus most variable
segments. Windows are computed with `potytools::rolling_alignment_stats()`
(300 bp window, 30 bp step, codon-aware).
